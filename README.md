JobBoard

A job listing application built with React that allows users to browse real job postings, filter by category and location, and bookmark jobs for later.

Live Demo
https://job-board-ui-uhl7-tau.vercel.app/

Features
- Browse real job listings from The Muse API
- Filter jobs by category and location type
- View full job details with description and apply link
- Bookmark jobs and persist them using localStorage
- Clean professional light UI
- Responsive grid layout

Tech Stack
- React 18
- React Router v6
- Custom Hooks (useFetch, useLocalStorage)
- The Muse Jobs API
- CSS Modules
- Deployed on Vercel

Run Locally
git clone https://github.com/m-ahmad-12/job-board-ui.git
cd job-board-ui
npm install
npm run dev
